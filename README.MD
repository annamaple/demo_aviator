## Java表达式计算工具-Aviator的使用

添加依赖:
```
<!-- https://mvnrepository.com/artifact/com.googlecode.aviator/aviator -->
<dependency>
    <groupId>com.googlecode.aviator</groupId>
    <artifactId>aviator</artifactId>
    <version>5.3.0</version>
</dependency>
```

Aviator 有两种常用的使用方式： 
- 直接在Java代码中使用(表达式运算)
- 在.av脚本文件中使用
